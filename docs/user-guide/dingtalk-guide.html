钉钉桥接配置指南

钉钉与桌面 Agent 双向对话，当前单聊 / 群聊现象已与飞书、企业微信对齐

钉钉 飞书 企业微信 微信
前提与配置
  1. 在钉钉开放平台创建企业内部应用并添加机器人能力
  2. 记录 AppKey、AppSecret、robotCode
  3. 如需主动发送，开通通讯录与机器人发消息权限
  4. 在 Hydro Desktop 的钉钉设置中填写配置并连接
如果桌面端主动发送列表为空，优先检查 robotCode 与通讯录权限。
当前行为
  • 单聊按用户维度维护会话
  • 群聊按群维度维护会话
  • 群聊不会再错误绑定到某个单聊用户
  • 主动发送绑定后，后续 IM 回复会优先回当前绑定会话
命令系统
命令说明
/help显示帮助
/status查看当前聊天上下文下的历史会话状态
/sessions列出活跃会话
/new [目录]新建会话
/resume [编号]恢复历史会话
/rename <名称>重命名当前会话
/close关闭当前会话

/close 当前不支持带编号参数。

状态标记
  • ✅ 当前映射会话
  • 🔵 已激活但不是当前映射
  • ⭕ 历史存在但当前未激活
