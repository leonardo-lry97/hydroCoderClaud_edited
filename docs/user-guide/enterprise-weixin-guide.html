企业微信桥接配置指南

企业微信与桌面 Agent 双向对话，保留原生流式回复能力

钉钉 飞书 企业微信 微信
前提与配置
  1. 在企业微信创建智能机器人，获取 Bot ID 与 Secret
  2. 开通消息收发与媒体文件上传权限
  3. 在 Hydro Desktop 的企业微信设置中填写配置并连接
如需桌面端主动给成员发消息，还需要额外完成 wecom-cli 初始化。
当前行为
  • 单聊按用户维度维护会话，群聊按群维度维护会话
  • 主动发送绑定后，后续 IM 回复会优先回当前绑定会话
  • 文本回复保留企业微信原生流式输出
  • 群名称展示可优先使用本地 alias / 已知群名
命令系统
命令说明
/help显示帮助
/status查看当前聊天上下文下的历史会话状态
/sessions列出活跃会话
/new [目录]新建会话
/resume [编号]恢复历史会话
/rename <名称>重命名当前会话
/close关闭当前会话

/close 当前不支持带编号参数。

状态标记
  • ✅ 当前映射会话
  • 🔵 已激活但不是当前映射
  • ⭕ 历史存在但当前未激活
