飞书桥接配置指南

飞书与桌面 Agent 双向对话，当前命令结果统一为文本 / Markdown 风格回复

钉钉 飞书 企业微信 微信
前提与配置
  1. 在飞书开放平台创建企业自建应用，获取 App ID 与 App Secret
  2. 开通机器人、发送消息、通讯录、图片相关权限
  3. 在 Hydro Desktop 的飞书设置中填写凭据并连接
飞书桥接使用 WebSocket 长连接接收入站，不需要配置 HTTP 回调 URL。
当前行为
  • 单聊按用户维度维护会话，群聊按群维度维护会话
  • 桌面端主动发送后，会把当前桌面会话绑定到该飞书目标
  • 后续该目标从飞书回复时，会优先回到当前绑定会话
  • 关闭当前会话后，再次入站会进入历史选择 / 新建逻辑
命令系统

发送以 / 开头的消息，会走命令链路，不进入 AI 对话。

命令说明
/help显示帮助
/status查看当前聊天上下文下的历史会话状态
/sessions列出活跃会话
/new [目录]新建会话
/resume [编号]恢复历史会话
/rename <名称>重命名当前会话
/close关闭当前会话

说明：飞书本轮已经移除交互卡片主方案，命令结果不再依赖卡片按钮。

状态标记
  • ✅ 当前映射会话
  • 🔵 已激活但不是当前映射
  • ⭕ 历史存在但当前未激活
